1. Introduction

Kfind looks for files matching search criteria within a directory hierarchy.

Often you know that a file is 'out there somewhere', but you can't remember where it is, even with a good effort. So KFind is exactly the tool for that problem. Having found the required file(s), you can:

  • save them,
  • delete them,
  • change their file properties,
  • store them to archives and/or
  • open the directory containing the file(s) in the file manager (Kfm).

I hope you find this program useful and I would appreciate any suggestions and comments.
