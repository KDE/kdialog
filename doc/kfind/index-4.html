4. The Menu Entries

4.1 File

The following functions are available from the file menu:

Open

This option calls kfm to open the currently selected file, using the default binding e.g Program files will be executed, text files will start the text editor, etc. This action can also be performed by double-clicking on the filename in the list.

Add to archive

Adds the selected file to an archive.

Delete

Deletes the selected file.

Properties

Allows the properties of the selected file to be altered.

Open Containing Folder

Opens the directory containing the selected file in a file manager (KFM) window

Save Search

Saves the list of selected files

Quit

Exit Kfind

4.2 Edit

Entries in this menu are disabled.

4.3 Options

From this menu you can start the preferences dialog. The first tab of the preferences dialog allows one to select the name and format of the file where search results are stored. The second tab shows available filetypes and archivers details.

Saving

Specify the file name where the the search results are stored. The default is .kfind-results.html in the user's home directory. If another name is chosen, one can specify the format of the stored results. HTML and Text (.txt) are the currently supported formats.

Archivers

This dialog allows viewing (and in the future, probably editing) of the archiver definitions used by Add to Archive function.

4.4 Help

>Show information about KFind and KFind documentation.
